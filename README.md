# 🔴 Sistema de Times Pokémon (API) - (Postman)

## 🤔 Sobre o projeto
Este projeto é um conjunto de testes de API desenvolvido com foco em estudo e prática da ferramenta Postman, incluindo organização de requisições, criação de coleções e automação de workflows.

O objetivo é aprimorar habilidades em testes de API, manipulação de requisições REST e validação de respostas em cenários reais de integração.

⚠️ **Aviso importante:** Pokémon é propriedade intelectual da Nintendo, Game Freak e The Pokémon Company. Todos os direitos sobre nomes, imagens e referências pertencem aos seus respectivos detentores. Este projeto não possui fins comerciais e é apenas para fins educacionais e de portfólio.

---

## 👨‍💻 Tecnologias utilizadas
- Postman  
- APIs REST  
- JSON  
- Workflows de automação
